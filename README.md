# Wolfsfera Solana Hub

Scaffolding inicial del hub de recursos de Wolfsfera para el ecosistema Solana. Incluye Next.js 14 con App Router,
TypeScript y TailwindCSS configurados.

## Requisitos

- Node.js 18 o superior
- npm 10 o superior
